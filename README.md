# AGENT-BTCFutures-Analysis-SOP

## 中文

这是一个面向 BTC 日内合约分析的 AI 辅助交易 SOP 与知识库，主要用于 Binance USD-M 永续合约的行情研判、交易计划、风险控制、复盘沉淀和版本迭代。

当前生效版本为 `BTC日内合约分析SOP_v3.5`，入口见：

- `00.项目管理/当前生效版本.md`
- `01.交易SOP/BTC日内合约分析SOP_v3.5.md`

v3.5 版本重点引入了更结构化的分析流程，包括多智能体多空辩论、证据包校验、评级约束、异常数据处理、复核机制和标准化输出模板。项目中的历史 SOP、策略研究、模板和归档资料用于追踪迭代过程，不会自动替代当前生效版本。

本项目适合与 Codex、TradingView MCP、Binance 合约数据接口及相关分析工具配合使用，可作为 AI 交易分析代理的流程规范、提示词基础和知识库框架。

## English

An AI-assisted trading SOP and knowledge base for BTC intraday futures analysis, focused on Binance USD-M perpetual futures, market analysis, trade planning, risk control, review workflows, and versioned strategy iteration.

The current active version is `BTC日内合约分析SOP_v3.5`. Main entry points:

- `00.项目管理/当前生效版本.md`
- `01.交易SOP/BTC日内合约分析SOP_v3.5.md`

Version 3.5 introduces a more structured analysis workflow, including multi-agent bull/bear debate, evidence-packet validation, deterministic grading constraints, abnormal-data handling, review layers, and standardized output templates. Historical SOP files, strategy notes, templates, and archives are kept for traceability and do not automatically replace the active version.

This project is designed to work with Codex, TradingView MCP, Binance futures data APIs, and related analysis tools. It can serve as a workflow specification, prompt foundation, and knowledge-base framework for AI trading analysis agents.
